== Description ==

Christian Jokes widget will fetch the latest or random joke from [christian-jokes.net](http://www.christian-jokes.net) website, and display them on your blog.

You can select between:<br />
- Latest Joke<br />
- Random joke<br />


Christian Jokes widget uses in-built MagPie for reading data and HTTP Snoopy Client for fetching data from [christian-jokes.net](http://www.christian-jokes.net).

For less stress on the system, Christian Jokes widget does not use database, but it caches data to a file.

* [Widget Homepage:](http://www.christian-jokes.net/tools/wordpress.aspx "Widget Homepage")
* [Live Demo](http://www.thequest4truth.com/ "Live Demo")

== Installation ==

1. Upload `christian-jokes` directory to the `/wp-content/plugins/` directory
2. Create `/wp-content/cache` folder if it doesn't exist and chmod it to 777
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to 'Design' -> 'Widgets' to add, and configure the widget
